# Job Application Tracker - Frontend

A React-based web application for tracking job applications.

## Features

- User authentication (Register/Login)
- Dashboard with statistics
- Add, view, update, delete applications
- Filter by status
- Search by company/position
- Responsive design

## Tech Stack

- React
- React Router DOM
- Axios
- Context API for state management

## Installation

1. Install dependencies:
bash
npm install


2. Start development server:
bash
npm run dev


Frontend runs on http://localhost:5173

## Usage

1. Register a new account
2. Login with your credentials
3. Add job applications
4. Track your progress with status updates
5. View statistics on your dashboard
